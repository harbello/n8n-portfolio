# n8n Portfolio

A collection of real-world automation workflows built with **n8n**, AI, APIs, and cloud services.

## Projects

| Project | Description |
|---------|-------------|
| 01 - Weather Dashboard | Automated weather report via email |
| 02 - Gmail Attachment Saver | Automatically save Gmail attachments to Google Drive |
| 03 - AI Email Classifier | AI categorizes incoming emails |
| 04 - PDF Invoice Organizer | Extract invoice data from PDFs |
| 05 - Daily Sales Report Generator | Generate daily sales summaries |
| 06 - AI Support Ticket Router | AI analyzes and routes maintenance complaints |
| 07 - AI Resume Analyzer | AI screens resumes and scores candidates |

## Technologies

- n8n
- Groq AI
- OpenAI
- Google Sheets
- Gmail
- Google Drive
- JavaScript
- REST APIs
