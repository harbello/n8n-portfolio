# Weather Dashboard

An automated weather reporting workflow built with n8n.

## Features

- Runs automatically on a schedule
- Retrieves live weather information
- Formats weather data
- Sends a weather report via Gmail

## Technologies

- n8n
- Weather API
- Gmail

## Workflow

↓

Get Weather Data

↓

Format Weather Response

↓

Send Email
